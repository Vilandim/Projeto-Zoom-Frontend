Desenvolvimento Web

Sobre este curso:

Bem vindo ao curso Desenvolvimento Web. Para iniciar o treinamento não é necessário nenhum conhecimento prévio na área, o aluno partirá do zero e ao final do treinamento alcançará um nível profissional.

Aprenda agora mesmo as tecnologias: HTML5, CSS3, BootStrap 5 e Java Script. Além disso, o aluno desenvolverá vários projetos reais. Aqui o aluno aprende praticando!

Ao final do curso você será capaz de criar aplicações WEB e integrar as principais tecnologias Web através de projetos reais. Tudo na prática!

Aulas:

  1. O que irei aprender? Preparando-se para o desenvolvimento
  2. IDEs e editores de texto de código fonte
  3. Criando o seu primeiro projeto no Visual Studio Code
